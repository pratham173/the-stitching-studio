## 🎨 Design Philosophy

The website is designed with a **minimal, feminine, and premium** aesthetic — reflecting the elegance of a high-end tailoring boutique.

### Color Palette

| Role | Color | Hex Code |
|------|-------|----------|
| Primary | Deep Pink / Magenta | `#d81b60` |
| Secondary | Soft Blush Pink | `#f8c6d8` |
| Background | Off-White / Light Grey | `#f5f5f5` |
| Text | Dark Charcoal | `#222222` |

### Typography

| Usage | Font | Style |
|-------|------|-------|
| Headings | **Playfair Display** | Elegant Serif |
| Body Text | **Poppins** | Clean Sans-Serif |

Fonts are loaded via [Google Fonts](https://fonts.google.com/) CDN.

## ✨ Features

- 📱 **Mobile-First Responsive Design** — looks great on phones, tablets, and desktops
- 🧭 **Sticky Navigation Bar** — always accessible with hamburger menu on mobile
- 🔄 **Smooth Scroll Navigation** — seamless transitions between sections
- 💬 **Floating WhatsApp Button** — instant customer communication
- 📝 **Appointment/Contact Form** — with client-side validation
- 🖼️ **Gallery with Hover Effects** — zoom animations on outfit images
- 🔍 **SEO-Friendly Structure** — semantic HTML5, meta tags, Open Graph tags
- ♿ **Accessible Design** — proper alt texts, ARIA labels, contrast ratios
- ⚡ **Fast Loading** — minimal dependencies, optimized CSS/JS

## 🚀 Deployment

This website is designed to be deployed instantly via **GitHub Pages**.

### Steps:

1. Go to your repository **Settings** → **Pages**
2. Under **Source**, select the `main` branch and `/ (root)` folder
3. Click **Save**
4. Your site will be live at: `https://pratham173.github.io/the-stitching-studio/`
